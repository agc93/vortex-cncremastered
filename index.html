C&C Remastered Collection Support for Vortex

Documentation GitHub Nexus Mods

Adds support for Red Alert and Tiberian Dawn mods to the Vortex mod manager. Fully open-source.

This is a simple extension to add Command and Conquer Remastered Collection support to the Vortex mod manager. With this extension you can install, remove and manage your Command and Conquer mods and Red Alert mods using Vortex, perfect for players already familiar with Vortex.

This project is unofficial! We have no association with Nexus Mods nor with EA/Petroglyph and is only a community project.

Full Vortex integration

Manage your C&C Remastered mods the same way you manage any other Vortex-supported game.

Contributions welcome!

The extension is fully open-source and open to community contributions and fixes!
