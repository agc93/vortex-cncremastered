{{< /blocks/cover >}} {{% blocks/lead color="primary" %}} This is a simple extension to add Command and Conquer Remastered Collection support to the Vortex mod manager. With this extension you can install, remove and manage your Command and Conquer mods and Red Alert mods using Vortex, perfect for players already familiar with Vortex. **This project is unofficial!** We have no association with Nexus Mods nor with EA/Petroglyph and is only a community project. {{% /blocks/lead %}} {{< blocks/section color="dark" >}} {{% blocks/feature icon="fa-tools" title="Full Vortex integration" %}} Manage your C&C Remastered mods the same way you manage any other Vortex-supported game. {{% /blocks/feature %}} {{% blocks/feature icon="fab fa-github" title="Contributions welcome!" url="https://github.com/agc93/vortex-cncremastered" %}} The extension is fully open-source and open to community contributions and fixes! {{% /blocks/feature %}} {{% blocks/feature icon="fas fa-cloud-download-alt" title="Supports Most Mods" %}} You should be able to install almost any mod for either Red Alert or Command and Conquer automatically. {{% /blocks/feature %}} {{< /blocks/section >}}
